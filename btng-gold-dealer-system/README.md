# BTNG Gold Dealer System
Live spot-synced buy & sell quotes. Automatic spreads. VAT-ready receipts. Team and branch management. All in one professional dealer system. No Excel. No pricing mistakes. No chaos at the counter. Built by the BTNG Gold API team.

## Features
- **Live Spot Sync**: Real-time gold prices from oracle API
- **Automatic Spreads**: Configurable buy/sell margins
- **VAT-Ready Receipts**: Ghana-compliant tax calculations (Act 1151)
- **Team Management**: Multi-user role-based access
- **Branch Management**: Multi-location inventory tracking
- **Professional UI**: Clean dashboard for counter operations

## Quick Start
```bash
npm install
npm run dev
# Open http://localhost:3000
```
